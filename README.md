## Usage

These prompts are designed for AI tools that support attachments and web browsing, including Claude, ChatGPT, Copilot, and Gemini.

Every prompt uses `[bracketed variables]` for the context you supply. Before running a prompt:

1. Replace all `[bracketed variables]` with specifics from your situation
2. Add any constraints the model needs (timeline, audience, technical context)
3. Remove any input fields you aren't using before running the prompt.
4. Treat the output as a first draft. Push back, ask follow-ups, refine.

## Contributing

Fork the repo, add your prompt to the relevant category file, and open a PR against `main`.

Each prompt must follow the standard template:

```
## [Title]
**Use Case:** ...
**Prompt:**
> ...
**Why It Works:** ...
```

PRs without a **Why It Works** rationale won't be merged. The bar is: does this prompt produce meaningfully better output than asking the question directly? If yes, explain why. If the answer is "it's just a good question," it doesn't belong here.
